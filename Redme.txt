
Перед установкой экспортировать рабочую базу данных ( При экспорте выбрать: Метод экспорта:Обычный - отображать все возможные настройки; Параметры создания объектов поставить галочку на против Добавить выражение CREATE DATABASE / USE)!

	Инструкция по запуску: 
1) Установить XAMPP из архива проекта на диск C (файл xampp-windows-x64-8.1.1-2-VS16-installer.exe)
2) В конфигурации xampp заменить порт c 80 на 8000 (путь к файлу C:\xampp\apache\conf\httpd.conf) (строчка Listen 8000)
3) Запустить xampp
4) В программе xampp запустить модули Apache и MySQL
5) Извлеч папку проэкта в папку установленной программы xapmm ( путь к папке xampp C:\xampp\htdocs)
6) Зайти в PhpMyAdmin (http://localhost:8000/phpmyadmin/) и импортировать старую базу данных
7) В коммандной строке перейти в корень проекта (cd C:\xampp\htdocs\intersection) и выполнить комманду php artisan migrate
Дополнение: если в коммандной строке не работает php необходимо прописать к нему путь
	Перейти в Мой компьютер->Свойства->Дополнительные параметры системы->Параметры среды->Системные переменные(создать новую переменную с именем Path и значением C:\xampp\php)

8) Запустить проект можно по аддресу http://ipкомпьютера:8000/intersection/public/home или http://localhost:8000/intersection/public/home
